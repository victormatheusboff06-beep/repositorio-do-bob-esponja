### **Resumo - IA26 Web Design**

**Objetivo da disciplina:**  
Ensinar os fundamentos de criação e estilização de páginas web usando **HTML**, **CSS** e noções básicas de **JavaScript**. É uma disciplina introdutória, mas essencial para quem quer seguir carreira em desenvolvimento web.

### **Antes de começar**
- Use **Visual Studio Code** como editor de código.
- Use **Google Chrome** como navegador.
- É importante ter o ambiente configurado.

### **HTML (HyperText Markup Language)**
- Não é linguagem de programação, mas sim **linguagem de marcação**.
- Serve para **estruturar** o conteúdo da página (títulos, parágrafos, links, imagens, etc.).
- Foco em **semântica** e **acessibilidade** (importante para leitores de tela e SEO).
- Usa **tags** com abertura e fechamento: `<tag>conteúdo</tag>`.
- Tags recomendadas: `<strong>` (negrito importante) e `<em>` (itálico com ênfase) ao invés de `<b>` e `<i>`.

**Estrutura básica de um documento HTML:**

```html
<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <title>Título da Página</title>
</head>
<body>
    <!-- Conteúdo visível aqui -->
</body>
</html>
```

**Atributos importantes:** `href`, `src`, `alt`, `class`, `id`.

### **CSS (Cascading Style Sheets)**
- Linguagem usada para definir a **aparência** e o **layout** da página.
- Separa o conteúdo (HTML) do visual (CSS).
- Conectado ao HTML geralmente por arquivo externo (`<link rel="stylesheet" href="styles.css">`).

**Sintaxe básica:**

```css
seletor {
    propriedade: valor;
}
```

**Principais tipos de seletores:**
- Por elemento: `h1 {}`
- Por classe: `.titulo {}`
- Por ID: `#principal {}`
- Descendente: `main section p {}`
- Pseudo-classes: `a:hover {}`

### **Principais conceitos**
- HTML = estrutura e significado do conteúdo.
- CSS = estilo, cores, fontes, espaçamento e layout.
- Boas práticas: usar tags semânticas, separar HTML do CSS e priorizar acessibilidade.
